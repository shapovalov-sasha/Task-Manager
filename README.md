# How to

Clone this repository in the desired folder:

`git clone git@github.com:shapovalov-sasha/Task-Manager.git`

Navigate into project directory:

`cd Task-Manager`

Install by running:

`npm install`

When installation is done, start the project in development mode:

`npm start`
